# Vingo Food Delivery 🍔

This is a full-stack food delivery web app I’m building using the MERN stack.  
Right now I’ve implemented authentication and started working with Redux Toolkit for state management.

## 🚀 Features (Work in Progress)

- User Authentication (JWT based)
- Google Sign-In (Firebase)
- Forgot / Reset Password
- User state management using Redux Toolkit
- Persistent login (fetch current user)

## 🛠 Tech Stack

- Frontend: React, Redux Toolkit
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT + Firebase (Google Auth)
